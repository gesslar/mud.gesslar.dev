Mudlet packages

To install these packages in Mudlet, simply drag and drop the links onto Mudlet. Ensure you're at least at Mudlet v4.13.1

  • Threshold Color Health Bar Percent.mpackage (requires settings percent on)
  • Repeat Last Command.mpackage (repeat the last command sent to the MUD via Ctrl-R)
  • ThresholdBeep.mpackage (play a sound when the game sends you a beep!)
  • ThreshKeepalive.mpackage (send a keepalive message every 2.5 minutes)
  • URLiser.mpackage (make links clickable again)
  • ThreshBuff.mpackage (display buffs/debuffs in Mudlet)
  • ThreshCopy.mpackage (copy functions for Threshold)
There's also a bloodjewel calculator, if you care.
